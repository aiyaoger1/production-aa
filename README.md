# 生产下单系统

一键自动化构建的独立EXE程序，无需安装Python。

## 获取EXE文件

1. 查看右侧 Actions → 最新构建
2. 点击构建记录
3. 在"Artifacts"处下载"生产下单系统-发布包.zip"

## 使用方法

1. 解压下载的ZIP文件
2. 双击"启动系统.bat"
3. 等待10-20秒程序启动
4. 打开浏览器访问: http://localhost:5000

## 手动触发构建

如果没有自动构建，可：
1. 进入 Actions 标签页
2. 选择 "Build Windows EXE" 工作流
3. 点击 "Run workflow"
